A. Description

With this script you can make nice and interactive drop down menus. The advantage of this script is that it not only gives you the ability to make list drop down menu. You can also use a div as drop down element. This way you can create big drop down menus like under the buttons products and tutorials in the live preview. Or you can create a login panel in your drop down like I did in the live preview.

C. Install

The first thing you need to do is including jquery and the dropmenu.js script (Script/js/dropmenu.js) in the head of your page. The dropmenu.js can be found under Script/js/. Copy and include the style.css from Example 3 (Script/Example3). This is a very basic css style that you very easily can customize to fit your needs.

1
<script src="jquery.js"></script>
<script src="dropmenu.js"></script>
<link rel="stylesheet" href="style.css" type="text/css" />

Next thing to do is adding the HTML of your dropdown menu. For example:

2
<ul id="nav-one" class="dropmenu">
<li><a href="#">Home</a></li>
<li>
<a href="#">Forum</a>
<ul>
<li><a href="#">Support</a></li>
<li><a href="#">Help</a></li>
<li><a href="#">Examples</a></li>
<li><a href="#">Your work</a></li>
</ul>
</li>
<li>
<a href="#">Downloads</a>
<ul>
<li><a href="#">Tools</a></li>
<li><a href="#">Office</a></li>
<li><a href="#">Custom projects</a></li>
</ul>
</li>
<li>
<a href="#">Downloads</a>
<div>This is a div with some content.</div>
</li>
</ul>

As you can see in the example above, you can use an ul or a div as dropdown element.

3

<script>
$(document).ready(function(){
$("#nav-one").dropmenu({openAnimation: "fade",
closeAnimation: "fade",
openClick: false,
openSpeed: 500,
closeSpeed: 700,
closeDelay:500,
onHide: function(){},
onHidden: function(){},
onShow: function(){},
onShown: function(){} });
});
</script>

D. Parameters(default value)

  • openAnimation: fade, size, slide
  • closeAnimation: fade, size, slide
  • openClick: A click on the arrow span is needed before the dropdown menu opens.
  • openSpeed: time of the open animation in miliseconds
  • closeSpeed: time of the close animation in miliseconds
  • closeDelay: delayed close time in miliseconds
  • onHide: Callback function when a submenu close animation starts (this is the submenu)
  • onHidden: Callback function when a submenu close animation ends (this is the submenu)
  • onShow: Callback function when a submenu open animation starts (this is the submenu)
  • onShown: Callback function when a submenu open animation ends (this is the submenu
  • zindex: With this parameter you can define wich is the lowest zindex that must be applied to the dropmenu. Leave this setting out if you don't know what zindex is.
  • autoAddArrowElements: If set to true there will automaticly be added a span element to an a elemen that has shildren. This is handy to show a nice arrow on parent menu elements.

E. Custom animation

With this you can create you own customized animations. The first thing todo is create an animation function.

1

<script>
function customOpenClose(){
jQuery(this).animate({
opacity: 'toggle',
width: 'toggle'
}, 500);
}
</script>

The "this" we used in the above code will refer to the ul/div element of the submenu. Next we define this function as our animation for the menu.

2

<script>
function customOpenClose(){
jQuery(this).animate({
opacity: 'toggle',
width: 'toggle'
}, 500);
}

$(".dropmenu").dropmenu({

...
openAnimation: customOpenClose,
closeAnimation: customOpenClose,
...
});
</script>

F. Drop menu on top of flash content

It's no problem to show drop menu on top of flash content or another div. To show it above another div you just need to make sure to set the zindex parameter higher than the z-index of the div you want show under the menu.

For flash content you add the following parameter to the object element:

<param name="wmode" value="transparent" />

and add this to the embed tag:

wmode="transparent"
