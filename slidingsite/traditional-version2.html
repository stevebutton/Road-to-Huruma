Welcome to FSS

FSS is a jQuery plugin for creating full screen sliding websites. This website is just a sample of what can be accomplished with FSS. It can be easily integrated into your website by placing a piece of javascript code between the <head></head> tags. FSS can work with any website, regardless of its design.

Each Slide Has a URL

FSS supports deep linking, which means that you can link to any specific part of the website from anywhere. Deep linking also lets you use the browser's back and forward buttons. Another cool thing about FSS is that the pages which are dynamically loaded by using AJAX also support deep linking.

Load Pages Dynamically

You can dynamically load the content into your website by using AJAX. All of the external content will be preloaded by default, however, this can be easily changed by using the plugin options. A gif preloader can be displayed during the loading process. The AJAX feature is optional, FSS can also work with a single index page.

Fully Configurable

FSS is a fully configurable jQuery slider which can be modified to suit your needs. The transition method (Horizontal/Vertical), transition speed and slide duration can be adjusted by using the plugin options. Other adjustable parameters include autoplay, looping, keyboard navigation and easing.
