Instructions

Mansion is a free multimedia theme for WordPress. The theme design was developed with users who appreciate visual browsing and prefer a minimal layout, with their photos and multimedia featured front and center. Mansion caters the photobloggers audience who love to just throw in the photographs they have and occasionally share their views as blog posts. Subscribe to Graph Paper Press' feed or follow us on the bleeding edge of development by following us on Twitter.

Here is a brief list of the major features:

  • Fully flexible according to the browser width
  • Choose to show one image per post or all on the grid
  • Pulls all the images you have in your library
  • Separate Blog archive grid
  • Highlight your posts by tagging it "featured"
  • Widgetized sidebar in single pages
  • Compatibility with WordPress 2.9+
  • Standards Compliance: This theme validates as Strict XHTML and has been tested on Firefox 3.6+, Safari 4.0+, Opera 10+, IE8, and IE7

Best Practices

If you want to make changes to how this theme looks and feels, then you should create a child theme. Never make changes to the original theme files. Why? Because when a new version of this theme is released, all of your changes will be lost when you update. Plus, making a child theme is incredibly easy. Here is how you do it.

  1. Installation

    1. Unzip mansion.zip to your computer's desktop. It should be contained within a folder called /mansion/
    2. Upload the folder and its contents to the themes directory of your blog via FTP, located in wp-content/themes where WordPress is installed.
    3. Click on Appearance - Themes menu from the WordPress admin panel and then activate Mansion.
    4. Click on View Site at the top of the page. Bam. Hot site!

    Note: Here's a related video about installing Modularity, another theme by Graph Paper Press

  2. Installation Troubleshooting

    If you've performed a clean install of Mansion and are having problems, make sure that the following conditions have been met:

    1. Make sure that you've installed Mansion properly. You should use an FTP program like FileZilla, WinSCP, or Fetch to upload your files. Do not use WordPress' Install a theme in .zip format option.
    2. Permissions: On most servers, Mansion files should be set to 644 and folders should be set to 755.
    3. Make sure that you've deactivated all of your plugins before installing and/or upgrading Mansion.
    4. Your Mansion folder should be named mansion. Do not rename this folder.
    5. If you are upgrading your version of Mansion, make sure to backup first and completely delete your old version of Mansion from your server before uploading the new version of Mansion to your server.

  3. Initial Setup - Media settings

    1. Click Settings -> Media. Set your thumbnails to 200 x 150. Set your medium photo size to 495 x 0. Set your large size to 950 x 0.
    2. Set your external video embed width to 495. Leave the height blank. This applied for your YouTube or Vimeo links.
    3. This theme will automatically create thumbnails from the images that you upload, when you upload them.
    IMPORTANT

    If you are migrating from an old theme to a new theme and your thumbnails look squished or distorted, you might need to re-upload the image you plan on using for the post thumbnail. This is because WordPress creates your image sizes based on the dimensions you specified above. Old thumbnails will not be automatically resized. You can also regenerate your thumbnails so that your old uploaded media have the new settings applied to it.

    By default, thumbnails are auto cropped by Wordpress. This theme checks first to see if you have added the URL to your prefered thumbnail to custom field called, you guessed it, thumbnail. If not, the theme will defer to the first photo that you uploaded using Wordpress' Add Photo icon. If you haven't uploaded a photo using the Add Photo icon, the theme will can the post body for an image, and use it. If no photo exists in the post body, the theme will use a default image. As you can see, this series of "checks" allow you to have 100% control over your post thumnbnails.

  4. Common problems

    This theme uses jQuery javascript to do lots of fancy stuff. If your theme looks broken, than you likely have a plugin that uses a conflicting javascript library. Deactivate your plugins, one-by-one, to determine which plugin is conflicting with jQuery.

  5. Frequently Asked Questions

    How do we get the Big image as in the homepage of your demo site?

    Tag your last post with "featured" tag and your image will automatically become big. Note it only happens for the last post.

    How do we get that blog look as in the demo?

    You will simply need to have a "blog" category and the blog archive page will automatically look like in the demo page. If you want to have other category also have this look, simply duplicate and rename the category-blog.php to category-catname.php where you will need to replace catname by your preferred category name.

    Why am I not getting the array of image as in the demo homepage?

    You can now set this from the Appearance -> Theme Options. How cool is that?
