How it works

The o'Slider plug-in works much like most other image sliders. It has Prev/Next arrows for navigation, auto run, bullets list or image thumbnails for navigation and other classic parameters. However the slider has a major distinction - it gives amazing live-background experience.

To achieve the needed effect the wrapper block, the sub layers for animation and the frame block for the images are required. On the plug-in call the wrapper is created from your main 'Div' block. All defined layers are placed and positioned inside this wrapper. The div with the images is placed on top of the layers.

The main wrapper block will have class name 'o-slider-external'. You are able to change its parameters (width, padding and margins) in o.slider.css file. Remember, if you are using several o'Slider blocks on a page and would like to have a different designs for each of them then you will have to use '#your_unique_slider_block.o-slider-external' in css file to add a unique style description.

The sub layers you would like to use for animation should be declared in css file (only the 'background' property is required) and listed in the function call as properties.

The frame div which wraps your images has a class name 'o-slider-frame'. You have to set up its width and height to the width and height of your images before running the slider.

Now you know about three main components: the 'o-slider-external' wrapper, the layers and the 'o-slider-frame' for the images. Follow the 'How to use' instructions and check the Examples page for more live examples.

How to use

First of all you have to include jQuery, the o'Slider script and the o'Slider CSS on your page:

<link rel="stylesheet" href="o.slider.css" type="text/css" media="screen" />
<script src="jquery.min.js" type="text/javascript"></script>
<script src="o.slider.min.js" type="text/javascript"></script>

Now you need to add some HTML. Create a Div block and place the images inside. All images in a sequence should be the same size. You can wrap an image with 'A' element to add a link to it.

<div id="block">
<img src="image_1.jpg" width="900" height="300" />
<img src="image_2.jpg" width="900" height="300" />
<a href="someurl.html"><img src="image_3.jpg" width="900" height="300" /></a>
</div>

Adding captions is a quite simple thing. Create 'Div' blocks with the captions outside of the main block and set its 'id' attribute to 'rel' of the related image. You have to remember to add a 'o-slider-captions' class name to the blocks.

<div id="block">
<img src="image_1.jpg" width="900" height="300" rel="caption_one" />
<img src="image_2.jpg" width="900" height="300" rel="caption_two" />
<a href="someurl.html"><img src="image_3.jpg" width="900" height="300" /></a>
</div>

<div class="o-slider-captions" id="caption_one">
<h3>First caption</h3>
<p>Lorem ipsum</p>
</div>

<div class="o-slider-captions" id="caption_two">
<h3>Second caption</h3>
<p>Lorem ipsum</p>
</div>

When the html is ready you have to set up CSS. Find some nice patterns for sub layers and add css blocks to the css file. Use a class name not an ID as the identifier.

.layer-one {
background: url('../images/image_one.png');
}
.layer-two {
background: url('../images/image_two.png');
}

Open o.slider.css file. Find '.o-slider-frame' block and change width and height to the size of your images. Now find '.o-slider-external' block which is the main container. By default it's 100% width, has no background and is centrally aligned. You can change the width, margins, paddings or the background according to your design.

And finally, Javascript. This short script will run the slider without animated layers.

$(document).ready(function(){
$('#block').oSlider();
});

To add animated layers add a 'layers' parameter which includes descriptions of all layers as shown in the example below. Use any name for the layer. Each layer must have 'className','offset' and 'direction' as parameters.

$(document).ready(function(){
$('#block').oSlider({
layers: {
first: {
style: 'layer-one',
offset: 5,
direction: 'ltr'
},
second: {
style: 'layer-two',
offset: 2,
direction: 'ltr'
},
some_next_level: {
style: 'some-next-layer-class-name',
offset: 26,
direction: 'ltr'
}
}
});
});

Except the 'layers' more parameters can be passed to the function. This is a more extended example.

$(document).ready(function(){
$('#block').oSlider({
autoStart: true,
pauseOnHover: true,
listNavThumbs: true,
arrowsNavHide: true,
layers: {
first: {
style: 'layer-one',
offset: 6,
direction: 'ltr'
},
second: {
style: 'layer-two',
offset: 16,
direction: 'ltr'
}
}
});
});

Parameters

Parameter Default Description
animSpeed 1000 Animation speed in milliseconds.
liquidLayout true Use it for a full-width liquid designs. If it's 'true' all positions will be recalculated on window re-size. If you have the fixed-width slider block then you should set parameter value to 'false' to avoid superfluous calculations.
autoStart false Set to 'true' if you would like to run the auto play mode.
pauseTime 3000 Pause time for auto play mode in milliseconds.
pauseOnHover false Set to 'true' if you would like to pause the auto play mode on mouse hover.
arrowsNav true Set to 'true' if you would like the Prev/Next arrows navigation to appear.
arrowsNavHide true If you set to 'true' Prev/Next arrows navigation will be hidden on mouse out.
listNav true Shows the list navigation. If 'listNavThumbs' is set to 'false' the bullets list will appear.
listNavThumbs false Use thumbnails of images instead of the bullets list.
listNavThumbName '_thumb' The ending should be added to the thumbnail file. For example if you have some 'image_file.jpg' then you have to create a smaller image file with the 'image_file_thumb.jpg' name. You can use any suitable file extension.
layers 'false' This parameter holds all sub layers and their parameters.
class - Class name of the sub layer.
offset - Usually animated sub layers move to lesser distance than the layer with images. You can choose the offset from 0 (not moving) to 100% (equal to images). Don’t put the ‘%’ mark when you set the variable.
direction - Direction for the animation of the sub layer. Can be either 'ltr' or 'rtl'.
