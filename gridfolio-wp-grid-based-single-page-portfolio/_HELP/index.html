Common Problems

  • Stylesheet Missing error?
    Do not upload the entire download you get from ThemeForest, just upload the theme folder itself
  • Custom field not working?
    They are case sensitive and must be spelled exactly, ex: categories not category. Also be sure to hit publish after making a change as "update" does not always take effect immediately.
  • Promo height issues

    The images being pulled into the promo must all be of the same height, even if they are auto-resized, upload images of the same proportions to ensure they're sized evenly.


Install Guide

Follow the steps in the order given below.
They are intended for a blank install but read through them even if switching a current site to learn how the site works.

  1. Start Blank
    Start with a blank installation, otherwise you'll have a mix of your posts and my import.
    Delete the default "About" page and hello world post.

  2. Adjust Media Settings Before Uploading Images (IMPORTANT)
    Before you add any images, you need to adjust the media sizes slightly.
    If you change these settings after adding images, you need to re-generate your thumbnails.
    1. In Settings > Media set Thumbnail to 140px by 140px (six 140px thumbnails fit perfectly in a gallery on a page)
    2. In Settings > Media set Medium to 620px by 620px (used for the portfolio slider)

     
  3. Import Sample XML File
    1. Go to Tools > Import > WordPress
    2. If prompted, install the importer plugin and activate it
    3. Choose the XML sample file included in the Sample Content folder, be sure to check the box to import sample files and attachments.

     

  4. Set the background image
    The background image/color is not set by default, this is to allow the use of the new Background tool in WordPress.
    In appearance > Background Choose a solid color or upload one included in the Sample Content folder.

  5. Items to update after the import
    1. In Settings > Reading > Front page choose the page titled: "Display All Pages"
      Display All Pages is a page template that pulls in all your pages and needs to be set as your home page.
    2. In Settings > Reading > Posts page choose the page titled: Blog
      Your blog page is optional and does not use a special page template.
    3. Add Widgets in Appearance > Widgets
      Widgets are used in the "Home" template and blog sidebar.
    4. Install Lightbox
      I suggest "FancyBox for WordPress" (Theme tested with version 2.7.2 by Jose Pardilla)
      Once this plugin has been activated, any link to an image will open in a lightbox.
    5. View and enter your settings in the Settings+ page
      Email address, etc...

     
Get into more detail on the individual page templates below


 

Display All Pages

The template that makes this a single page template is the page you assign the "Display All Pages" page template.

You NEED to assign this page as your home page in settings > reading, it will then be hidden from the main menu and not pull itself in.

This is the page that will grab and display all pages on your site, it's title and content are not displayed on your site.

If you need to re-arrange the pages as the display, use "page order" along the right when adding/editing other pages, lower numbers appear first (left).

Home / Top Section

The top section in the demo is using the page template "Home". It is NOT set as the home page under settings > reading. (See "Display All Pages" template above).

This page template pulls in the four home widgets. Drop 4 text widgets into Appearance > Widgets - Home Column A, B, C and D.

This "home" template also hides the page title, useful for the top section.

You do not actually need to use this template if you are using a different style as your home such as the portfolio, gallery, contact or default page template.

Portfolio Container

The portfolio section refers to the large items with an optional slider (if more than 1 image is used).

Create a page using the "Portfolio" page template, add a title some optional content. This will create a section/container to pull in your portfolio items added next.

Portfolio Items

Please create a page using the portfolio page template first to pull in these items. (Above)

The individual portfolio items are a custom post type included with this theme called Portfolio.

Click the Portfolio heading in the left menu and click "Add New".

Enter a title, a small amount of content and a sub heading (additional options box below the main content area).

To attach an image, click the "Add an image" square icon right above the main content area.

  1. Upload a few images but do not insert them into the post it will pull them in automatically.
  2. Click save all changes after they appear in the pop-up after uploading and close the pop-up
  3. They will be resized automatically to the "medium" setting you adjusted above BUT for the slider to function nicely, be sure they are the same aspect ratio.

Blog Summary

Create a blog summary that will appear on the single page layout.

Add a page, I'll call mine "Latest News".

Set it to use the page template: Blog Summary.

Additional settings for this are found in the Settings+ page.

Blog

(Full blog page)

Create page: Blog (leave as default page template)

Set your blog page in Settings > Reading > Posts Page (otherwise it will try to appear within the main content which is what the Blog Summary template is for)

This will be your full blog, which will open in a second page.

Additional settings for this are found in the Settings+ page.

Sidebar: add widgets in Appearance > Widgets

Gallery

The gallery section / effect is actually a standard page with the default WordPress gallery shortcode inserted.

Create a page, let's call ours Gallery.

Leave the template set to "Default template"

To attach an image, click the "Add an image" icon right above the main content area.

  1. Upload a few images but do not insert them into the post.
  2. Click save all changes after they appear in the pop-up after uploading
  3. You should now be in the gallery tab, if you have uploaded your images correctly it will say Gallery (3) where 3 is the number of images you added.
  4. While on this gallery tab, look at the gallery settings section and set the following:
    Link thumbnails to: Image File
    Gallery Columns: 6 (IMPORTANT)
  5. Insert the gallery into the post.

Contact

The contact form is inserted automatically when you create a page using the "Contact Form" page template.

Set the "to" email address in the Settings+ page.

Main Menu

Pages are added to this area automatically as you add and order pages.

If you have set the "Display all Pages" page template up properly and have it set as your home page, the menu items will all have #go-pageslug in the link which will scroll to a specific spot on the page.

See the settings+ page for excluding the blog from this menu.

Footer Menus

Two menus are defined for this in the appearance > Menus area allowing you to add any URL you want in the footer.

You can also disable the footer in the Settings+ page.

Shortcodes

Shortcodes are a very powerful way to insert custom functions directly into the main content of a wordpress page or post. They're items surrounded with square brackets like [shortcode] and can accept parameters (options) and/or content. Enter shortcodes in HTML mode on your WordPress editor, they are case sensitive.

You can override the default settings by referring to the settings below.
For example on cudazi_contact, I will set a "to" address: [cudazi_contact to="example@example.com"]

[cudazi_contact] - Used to display a contact form with PHP and JavaScript validation.

Note, this is inserted automatically into the Contact Form page template but you can insert it into a normal page template too.
Parameter Values Description
to you@youremail.com Must be populated with your email address.
subject Contact Form Message The subject of the email that you will get.
button Send Message Text of the send button
sent Message Sent! Message displayed to the user when a message is sent.
error Error, please be sure you filled out all fields properly. Error message user will see if filled out improperly.

[cudazi_column] - Used to create columned layouts, not needed for general users.

Note, see the sample layout .txt files created in the Sample Content folder.
Enter them exactly like the sample content file with the extra spaces above/below each element.
Parameter Values Description
width 3 1-12, corresponds to the grid_xx in the 960.gs Grid.
class alpha or omega For the first column, use alpha.
For the last column, use omega.
Leave blank for all others.
special boxed Use boxed to give rounded corners, a white background and shadow like the demo.

HTML Structure

This theme is a fixed layout with 1 to 12 columns. The general concept behind the 12 column framework is that all the columns need to add up to 12. In the single column sections, you will just see a div with a class of "grid_12", for a page with a main column and a sidebar, the classes add up to 12. learn more and find demos on their website...

If you are using nested columns, the first column class must also include "alpha" and the last column of "omega".

<div class="container_12">
    <div class="grid_12">
        Full width Column content.
        <div class="grid_6 alpha">nested first column</div>
        <div class="grid_3">nested second column</div>
        <div class="grid_3 omega">nested last column, adding up to 12.
    </div>
</div>

NetTuts has a great overview of the 960.gs Framework you may want to check out.

CSS Files and Structure

  • /style.css - Required by WordPress, unused for styling
  • /css/screen.css - This is the main CSS file. Site color information is in the colors-xxxxx.css files below.
    • /css/style-theme-default.css - Lite Version of the theme, applied after the main /css/style.css file.
    • /css/style-theme-dark.css - Dark Version of the theme, applied after the main /css/style.css file.
  • /css/960.css - 960.gs Framework, no need to edit
  • /css/dynamic-css.php - Outputs CSS from Settings+ page

See the Settings+ page area on how to "add on" css without having to edit the CSS directly - perfect for small changes.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

Please note that jQuery/JavaScript can apply classes after the page has loaded, to dig in further download firebug.
